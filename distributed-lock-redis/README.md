## 使用方式：
    
```java
    /*
    * reentrant:是否设置为可重入 默认为false
    */
    @Lock(reentrant = false)
    public void demo(){
        try {
            // ...业务逻辑
        } catch (InterruptedException e) {
            e.printStackTrace();
        }
    }
```
